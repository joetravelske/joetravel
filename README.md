# Joeventure Tours & Travel

Official website for Joeventure Tours - Kenya Safari Experiences

## Quick Deploy to Vercel

1. Push this repo to GitHub
2. Go to vercel.com
3. Click "Import Project"
4. Select this repo
5. Click "Deploy"

That's it! No configuration needed.
